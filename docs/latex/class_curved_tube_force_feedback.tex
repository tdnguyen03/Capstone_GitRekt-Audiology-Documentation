\doxysubsection{Detailed Description}
Provides force feedback for a curved tube object. 

This class calculates force feedback based on the cursor\textquotesingle{}s position, velocity, and radius relative to a spline-\/based curved tube. It also supports visualization of the tube and its closest points.
